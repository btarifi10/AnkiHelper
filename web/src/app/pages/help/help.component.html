First things first

Before you can use this app, ensure that Anki is running and that the Anki Connect add-on is installed. If it isn't, click the link and follow the instructions to downloading it. Once the add on is installed, click Tools > Add-ons to configure the add on. AnkiConnect should be on the list - click it and then click the Config button. All this does is allow the site to communicate with Anki.

Also, this app only works with Microsoft Word or Excel documents that have a .docx or .xlsx file extension. This shouldn't be a problem unless you are using a very old version of Word. If you are, and have .doc file extension, you can use an online tool like this one to convert it.

Once you have started, you will need to follow three steps in order to create cards using this app.


Step 1

The first step is to choose the deck that you will be adding cards to. There are three options here:

  • Choose a deck from the list of decks presented. This will mean that all notes processed from files will be added to this deck.
  • Read the deck from the Word document. This will mean each group in your Word document will need to have the deck name in the second line, after the number of items in the group. Note that each group will need to be assigned to one deck at most, but you can have multiple groups within the same Word document if you select this option. If the deck name given does not exist, it will be created.
  • Create a new deck. This is the same as the first option except a new deck will be created.

Files and templates

Step 2

Step 2 involves selecting and uploading any number of .docx files which will be scanned and analysed for cards/notes to be created. Simply upload the files one by one and click on the button to proceed.

Templates

For each group of notes you wish to create in Anki, a template has to be followed. Currently, this app supports templates which create notes of three models: Cloze, Basic, and Basic with reversed.

In each template, the first line of the group starts with a keyword, and the second line includes the number of items as well as the optional deck (see Step 1).

Check the example files to get a quicker understanding of the formats.

Cloze notes

The Cloze note template follows a "Name and describe" format. As such, the first line of a group has the format:
Name and describe the [title/heading of card].

The second line will have a number, an optional letter "M", and then the deck name if necessary:
  • The number indicates how many items are present, each separated by pressing the "enter/newline" key in word
  • The M, if there should be more than two cloze deletions (more on this).
  • The optional deck name
The following N lines (where N is the number in the second line) are of the format
[name]: [description item 1]; [description item 2]; ... ; [description item n]
If the "M" is not present, then a cloze note with all the names missing would be created, as well as one with all the descriptions (so two cards would be created). If the "M" is present, a first cloze note with be created with all the names missing, and one subsequent cloze note for each item's description (so N+1 cards would be created). This option is intended for longer, more detailed descriptions while keeping the bigger picture.

Basic, and Reversed
In the case of Basic, or Basic and reversed, the format is simpler. The first line contains either the word "Basic" or "Reversed", and the second line contains the number of items as well as the optional deck input. Following this, the two sides of each card are to be separated by a line with single foward slash ("/") and each card is separated by a line with a double foward slash ("//").

Step 3

All that's left to do is to review the cards being made and hit the confirmation button. Any error messages will be shown in this step.

Note that this app allows for text-only card creation at this point in time. If you wish to add images, or format the cards, you may do so after they are created.
